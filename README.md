# hrw-hashing

 highest random weight (HRW) hashing

## Installation

### Using Opam

```bash
opam install inquire
```

### Using Esy

```bash
esy add @opam/inquire
```

## Usage

### In OCaml

```ocaml
let () = Hrw_hashing.greet "World"
```
